# RAG Boot Sequence Minigame

Static HTML/CSS/JS minigame that teaches rag2f core components.

## Open locally

Open `docs/minigame/index.html` directly in a browser (file:// works).

## Embed in docs (iframe)

```html
<iframe
  src="/docs/minigame/index.html"
  title="RAG Boot Sequence"
  style="width: 100%; height: 720px; border: 0;"
  loading="lazy"
></iframe>
```

## Link from MkDocs or any site

Use a relative link to the static file:

```
/docs/minigame/index.html
```

If you want it inside MkDocs navigation, add a custom link in `mkdocs.yml` pointing to the file.
